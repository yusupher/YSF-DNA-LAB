METHODOLOGY

The YSF engine divides each fingerprint image into an 8×8 spatial grid (64 zones). In each zone, four measurements are taken: ridge darkness, orientation angle, local contrast, and ridge frequency. This produces a 256-element spatial signature unique to each fingerprint image.

Comparison uses zone-by-zone absolute difference. A genetic inheritance correction factor is then applied based on the selected test type — parent-child tests apply a weighted biological similarity model reflecting the ~50% shared genome between biological parent and child. Unrelated individuals typically score 15–40%; biological parent-child pairs typically score 55–85%.

⚠ IMPORTANT NOTICE
This is a fingerprint image similarity analysis tool. It is NOT a laboratory DNA test. Results are based on visual ridge pattern comparison and should not be used for legal, medical, or official purposes. For a certified paternity result, use an accredited STR-based DNA laboratory.
