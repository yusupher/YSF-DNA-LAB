METHODOLOGY

The YSF Image Analysis Engine extracts pixel-level intensity data from each fingerprint image and divides it into a 16×16 analysis grid. For each grid cell, statistical features including mean luminance, standard deviation, gradient orientation, and local contrast are computed.

Eight biometric parameters are derived: Ridge Density (pixel darkness distribution), Orientation Consistency (gradient alignment), Contrast Ratio (light/dark boundary sharpness), Core Region Intensity (central pattern strength), Edge Complexity (border irregularity), Spatial Frequency (ridge spacing), Symmetry Index (bilateral similarity), and Texture Entropy (pattern randomness).

Each feature is independently compared between the two images using normalized cross-correlation. The weighted average produces a final similarity percentage. The same image will always produce the same result — results are deterministic and image-dependent.

⚠ IMPORTANT NOTICE
This analysis is based on visual fingerprint image comparison and is not equivalent to laboratory STR-based DNA testing. Results reflect fingerprint ridge similarity only. For legal or medical paternity confirmation, please use an accredited DNA testing laboratory.
